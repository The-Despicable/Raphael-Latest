### 🚀 **Concrete Example (Practical)**  
**Goal**: Build a low-latency QA tool with strict token limits.  
**Implementation**:  
```python
# 1. Pre-filter queries (O(1) regex check)  
if len(prompt.split()) > 50:  
    prompt = prompt.split()[:50] + ["..."]  # Token-efficient truncation  

# 2. Call model with forced constraints (Speed + Token Efficiency)  
response = litellm.completion(  
    model="gpt-4o-mini",  
    messages=[{"role": "user", "content": prompt}],  
    max_tokens=128,          # Token efficiency  
    timeout=1.0              # Speed impact control  
)  

# 3. Post-process (Practicality)  
answer = response.choices[0].message.content[:200]  # Hard cap for UI fit  
```
